Options
---------

The icons named icon-* are the default icons used by the MySite Content Browser.  You may change
these to suit the design of your site.  Note that the MySite Icons module sets a default size for icons.
The provided icons are all 120x60 pixels.  If you change the default size, the presentation may not be optimal.
